# Reflow & Repaint

<b>Repaint</b>

- 레이아웃에는 영향을 주지 않지만, 가시성에는 영향을 주는 영향을 주는 엘리먼트 변경시 발생
  ex) <code>opacity</code>, <code>background-color</code>, <code>visibility</code>, <code>outline</code>

<b>Reflow</b>

- 모든 엘리먼트의 위치와 길이 등 다시 계산하는 것 문서의 일부 혹은 전체를 다시 렌더링 하는 것.
